Contenidos

En esta clase vamos a trabajar tips para implementar las siguientes tcnicas en R:

  • Generacin de un corpus de documentos
  • Nociones de pre-procesamiento de texto
  • Representacin mediante matrz de trmino-documento
  • Anlisis grfico de los trminos del corpus

Documentos para la demostracin

Para los ejemplos de esta presentacin vamos a utilizar los siguientes “documentos”:

doc1 = "El tango es un gnero musical y una danza caracterstica de la regin del Ro de la Plata"

doc2 = "El tango revolucion el baile popular introduciendo una danza sensual con pareja abrazada que propone una profunda relacin emocional de cada persona con su propio cuerpo y de los cuerpos de los bailarines entre s. Refirindose a esa relacin, Enrique Santos Discpolo, uno de sus mximos poetas, defini al tango como un pensamiento triste que se baila. 9"

docs = c(doc1, doc2)

Generacin del corpus de documentos

Vamos a utilizar la librera tm (Text Mining Package) de R: 

library(tm)

# Incorporamos los documentos a lo que ser nuestro Corpus (encodeado utf8)
# A VectorSource interprets each element of the vector x as a document.
corpus = Corpus(VectorSource(enc2utf8(docs)))

# Recuperamos el primer documento
inspect(corpus[1])

<<SimpleCorpus>>

Content:  documents: 1

Preprocesamiento (texto a minsculas)

Siempre es aconsejable reducir el espacio de dimensiones, por eso vamos a aplicar algunas transformaciones en el texto. La primera es pasarlo a minsculas:

# Convertimos el texto a minsculas
corpus.pro <- tm_map(corpus, content_transformer(tolower))

inspect(corpus.pro[1])

Matrz Trmino-Documento

Una vez preprocesado el texto, podemos generar la matrz trmino-documento:

# Pasamos como parmetro la mtrica de ponderacin de los trminos
# weightBin (pesado binario): marca presencia o ausencia
# weightTf (frecuencia del trmino): apariciones en el doc
dtm <- TermDocumentMatrix(corpus.pro, 
                          control = list(weighting = "weightTf"))

# Resumen de la dtm (document-term matrix)
dtm

<<TermDocumentMatrix (terms: 35, documents: 2)>>
Non-/sparse entries: 37/33
Sparsity           : 47%
Maximal term length: 14

dtm es un objeto especial de tipo 'term-document matrix'. Ahora vamos a pasar los datos a una matriz, para trabajar de forma ms cmoda:

Matrz Trmino-Documento (+++)

Ahora calculamos los trminos ms utilizados en todo el corpus

# Calculamos la frecuencia de cada trmino en el corpus
freq_term <- sort(rowSums(matriz_td),decreasing=TRUE)

# Generamos un dataframe con esta sumatoria (de rows)
df_freq <- data.frame(termino = names(freq_term), frecuencia=freq_term)

# Reseteamos el ndice (sino era el trmino "dueo" de la frecuencia)
row.names(df_freq) <- NULL

freq_term <- sort(rowSums(matriz_td),decreasing=TRUE)
df_freq <- data.frame(termino = names(freq_term), frecuencia=freq_term)

hist(df_freq$frecuencia, xlab='Cantidad de trminos', ylab='Frecuencia observada', main='Histograma')
